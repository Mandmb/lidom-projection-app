# League Projection Hitter Grader - Version 3

Adds a league dropdown so the model changes depending on the selected league.

Included models:
- LIDOM
- PR League
- MLB
- MiLB / AAA
- NPB
- KBO
- Mexican League
- Custom

Expected columns:
- playerFullName
- ForwVel
- ExitVel
- LaunchAng
- OPS vs FB95
- Contact%
- Hard Hit%
- Pull%

Run:
```bash
pip install -r requirements.txt
streamlit run app.py
```
